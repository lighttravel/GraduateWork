## 🔍 详细接线表

### I2S音频接口 (数字音频数据传输)

| 小智模块引脚 | ESP32-S3引脚 | 线缆颜色 | 功能说明 |
|-------------|-------------|---------|---------|
| **BCLK** | GPIO15 | 🟡 黄色 | I2S位时钟 (Bit Clock) |
| **LRCK** | GPIO16 | 🟢 绿色 | I2S左右声道时钟 (Word Select) |
| **DIN** | GPIO17 | 🔵 蓝色 | I2S数据输入 (麦克风 → ESP32) |
| **DOUT** | GPIO18 | 🟣 紫色 | I2S数据输出 (ESP32 → 扬声器) |
| **MCLK** | GPIO38 | ⚪ 白色 | I2S主时钟 (可选，推荐使用) |

### I2C控制接口 (ES8311编解码器控制)

| 小智模块引脚 | ESP32-S3引脚 | 线缆颜色 | 功能说明 |
|-------------|-------------|---------|---------|
| **SCL** | GPIO8 | 🟠 橙色 | I2C时钟线 (Clock) |
| **SDA** | GPIO9 | 🟤 棕色 | I2C数据线 (Data) |

### GPIO控制接口 (功放/模式控制)

| 小智模块引脚 | ESP32-S3引脚 | 线缆颜色 | 功能说明 |
|-------------|-------------|---------|---------|
| **SD_MODE** | GPIO10 | 🔴 红色 | 麦克风/耳机切换 (HIGH=麦克风, LOW=耳机) |
| **PA_EN** | GPIO11 | ⚫ 黑色 | 功放使能 (HIGH=开启, LOW=关闭) |
| **LED+** | GPIO47 | ⚪ 灰色 | 状态LED正极 (LED-接GND) |

### 电源接口

| 小智模块引脚 | ESP32-S3引脚 | 线缆规格 | 功能说明 |
|-------------|-------------|---------|---------|
| **VDD** | 3.3V | 🔴 红色 (≥22AWG) | 电源正极 (3.3V ±5%) |
| **GND** | GND | ⚫ 黑色 (≥22AWG) | 电源地 (与ESP32共地) |

## 📝 接线顺序

### 第一步: 电源连接 (必须先接!)
1. **GND** (黑色粗线) - 小智模块GND → ESP32-S3 GND
2. **VDD** (红色粗线) - 小智模块VDD → ESP32-S3 3.3V

### 第二步: I2S音频接口
3. **BCLK** (黄色) - GPIO15
4. **LRCK** (绿色) - GPIO16
5. **DIN** (蓝色) - GPIO17
6. **DOUT** (紫色) - GPIO18
7. **MCLK** (白色) - GPIO38

### 第三步: I2C控制接口
8. **SCL** (橙色) - GPIO8
9. **SDA** (棕色) - GPIO9

### 第四步: GPIO控制信号
10. **SD_MODE** (红色) - GPIO10
11. **PA_EN** (黑色) - GPIO11
12. **LED+** (灰色) - GPIO47

## ⚙️ 硬件配置检查清单

接好线后,请逐一检查:

- [ ] **电源极性正确**: VDD接3.3V, GND接GND (反接会烧毁模块!)
- [ ] **共地**: 小智模块GND与ESP32-S3 GND连通
- [ ] **无短路**: 用万用表测量VDD-GND阻抗 > 1kΩ
- [ ] **I2S接线正确**: DIN(麦克风入) ↔ GPIO17, DOUT(扬声器出) ↔ GPIO18
- [ ] **I2C地址**: ES8311默认I2C地址为0x18 (确认模块未改动)
- [ ] **功放连接**: 扬声器接到模块的SPK+/SPK-端子

## 🚨 常见接线错误

### ❌ 错误1: DIN/DOUT接反
```
错误: DIN(麦克风) → GPIO18, DOUT(扬声器) → GPIO17
正确: DIN(麦克风) → GPIO17, DOUT(扬声器) → GPIO18
```

### ❌ 错误2: I2C接反
```
错误: SCL → GPIO9, SDA → GPIO8
正确: SCL → GPIO8, SDA → GPIO9
```

### ❌ 错误3: 电源电压错误
```
错误: VDD接5V
正确: VDD接3.3V (模块不支持5V!)
```

### ❌ 错误4: 忘记共地
```
错误: 小智模块独立供电,未接GND到ESP32
正确: 小智模块GND必须与ESP32-S3 GND连通
```
